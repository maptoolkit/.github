# Maptoolkit — Geo APIs for Developers

Map APIs built on [OpenStreetMap](https://www.openstreetmap.org/) data. Display interactive maps, calculate routes, geocode addresses, generate static images, and more — GDPR-compliant, hosted in Europe.

**[Get started →](https://docs.maptoolkit.net)** · **[Book a demo →](https://www.maptoolkit.com)**

---

## APIs

| API | Description | Docs |
|-----|-------------|------|
| **Vector Tiles** | Interactive maps for MapLibre GL and Mapbox GL | [→](https://docs.maptoolkit.net/vector-tiles/) |
| **Raster Tiles** | Map tiles for Leaflet and OpenLayers | [→](https://docs.maptoolkit.net/raster-tiles/) |
| **Routing** | Turn-by-turn directions for car, bike, foot, and transit | [→](https://docs.maptoolkit.net/routing-api/) |
| **Geocoding** | Address lookup and coordinate resolution | [→](https://docs.maptoolkit.net/geocoding-api/) |
| **Elevation** | Elevation data for coordinates and route profiles | [→](https://docs.maptoolkit.net/elevation-api/) |
| **Isochrone** | Reachable areas within a given travel time | [→](https://docs.maptoolkit.net/isochrone-api/) |
| **Map Matching** | Snap GPS tracks to the road network | [→](https://docs.maptoolkit.net/map-matching-api/) |
| **Static Maps** | Server-side map image generation (screen-res, Retina, printable) | [→](https://docs.maptoolkit.net/static-maps-api/) |
| **Weather** | Weather forecasts and map overlays | [→](https://docs.maptoolkit.net/weather-api/) |
| **Data Connectors** | Serve your own data as vector tiles | [→](https://docs.maptoolkit.net/data-connectors/) |

---

## Why Maptoolkit

- **EU-hosted & GDPR-compliant** — no tracking, no third-party data sharing
- **OpenStreetMap-based** — open data, transparent coverage, community-driven updates
- **Built for developers** — clean REST APIs, modular JS library, comprehensive docs
- **Flexible deployment** — cloud API or self-hosted Maptoolkit Server (Kubernetes-ready)
- **Tailored support** — direct access to the team, not a ticket queue
